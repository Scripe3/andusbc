# andusbc (Android USB Controller)
<img src="images/logo.png" alt="Logo" width="200"/>

andusbc is an Android USB controller used for Android devices with malfunctioning screens, non-functional keys, inability to take
screenshots, or other issues. It doesn't have to be a problem; it can be used on devices without any issues.

This readme contains information about the installation and usage of this program. Information about different version releases
will also be provided.

## Setup

### Windows

Install the Android Platform Tools and Git:
```bash
winget install Google.PlatformTools Git.SDK
```

Clone this repo:
```bash
git clone --branch v1.0 --single-branch https://github.com/Scripe3/andusbc.git
cd andusbc
```

And enjoy!

#### Info:
After some time, you may no longer need to use Git clone:
```bash
winget install Scripe.Andusbc.v1.0
```
However, this is not currently applicable.
